# Microsoft Graph – Grundlagen

## Was ist Microsoft Graph?
Microsoft Graph ist die zentrale API, um auf Daten und Dienste in Microsoft 365 zuzugreifen:
- Benutzer
- Gruppen
- Dateien
- Teams
- SharePoint
- Intune
- Security

## Authentifizierung
Graph nutzt OAuth 2.0:
- Delegated Permissions (Benutzerkontext)
- Application Permissions (App-Kontext)

## Beispiel-Endpunkte
- GET /me
- GET /users
- GET /sites
- GET /groups

## Warum wichtig?
Graph ist das Rückgrat jeder Automatisierung und jedes modernen M365-Engineerings.
