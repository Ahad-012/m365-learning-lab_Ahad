# SharePoint Site Architecture – Grundlagen

## Site-Typen
- Team Sites (M365 Groups)
- Communication Sites
- Hub Sites

## Wichtige Konzepte
- Site Collections
- Subsites (Legacy)
- Libraries & Lists
- Permissions (SharePoint vs. M365 Group)

## Best Practices
- Keine Subsites mehr verwenden
- Jede Abteilung = eigene Site Collection
- Hub Sites für Navigation & Governance
- Sensitivity Labels für Sites nutzen
