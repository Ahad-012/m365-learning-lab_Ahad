# M365 Governance – Grundprinzipien

## Identität
- Alle Zugriffe über Entra ID
- MFA Pflicht für Admins

## Daten
- Klassifizierung über Sensitivity Labels
- Minimale Berechtigungen (Least Privilege)

## Geräte
- Nur compliant devices für Zugriff auf sensible Daten

## Überwachung
- Audit Logs aktiv
- Regelmäßige Review von Admin-Rollen
