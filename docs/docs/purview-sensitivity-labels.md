# Purview Sensitivity Labels – Design für M365 Lab

## Ziel
Schutz sensibler Daten durch Klassifizierung und Verschlüsselung.

## Geplante Labels
1. Öffentlich – Keine Einschränkungen
2. Intern – Nur für Mitarbeiter
3. Vertraulich – Verschlüsselung + eingeschränkter Zugriff
4. Vertraulich – HR – Nur HR-Gruppe
5. Vertraulich – Finance – Nur Finance-Gruppe

## Einstellungen
- Encryption: Aktiv für vertrauliche Labels
- Content Marking: Header + Footer
- Auto-Labeling: Für HR- und Finance-Daten geplant
