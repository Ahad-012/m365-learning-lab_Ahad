# SharePoint Site Design for M365 Lab

## Ziel
Eine klare, skalierbare Struktur für Abteilungen und Projekte.

## Sites
- /sites/HR – vertrauliche HR-Daten
- /sites/Finance – Finanzdokumente
- /sites/IT – interne IT-Dokumentation
- /sites/Sales – Vertriebsunterlagen
- /sites/Projects – Hub Site für Projektteams

## Governance
- Keine direkten User-Berechtigungen
- Nur Gruppen (M365 Groups oder SharePoint Groups)
- Sensitivity Labels für vertrauliche Bereiche
