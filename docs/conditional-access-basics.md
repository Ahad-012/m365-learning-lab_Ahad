# Conditional Access – Grundlagen

## Ziel
Conditional Access steuert den Zugriff auf Cloud-Ressourcen basierend auf:
- Benutzer
- Gerät
- Standort
- Risiko
- App

## Wichtige Elemente
- Conditions
- Grant Controls
- Session Controls

## Typische Policies
- MFA erzwingen
- Blockieren unsicherer Standorte
- Nur compliant devices zulassen
