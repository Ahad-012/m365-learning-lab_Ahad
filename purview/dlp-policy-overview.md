# Data Loss Prevention (DLP) – Überblick

## Ziel
Verhindern, dass sensible Daten das Unternehmen verlassen.

## DLP erkennt:
- Kreditkartennummern
- IBAN
- Personalausweisnummern
- Gesundheitsdaten
- Custom Sensitive Info Types

## Policy-Elemente
- Locations (Exchange, SharePoint, Teams, Devices)
- Conditions
- Actions (Block, Encrypt, Notify)
- User Overrides

## Beispiel
Blockiere Upload von „Vertraulich“-Dokumenten zu Dropbox.
